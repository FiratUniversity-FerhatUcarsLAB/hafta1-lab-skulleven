 Elbette! NES'e çıkan Super Mario Bros oyununda 1-1 bölümü, oyuncuların yüksek skor elde etmek için çeşitli teknikler kullanabildiği ikonik bir bölümdür. Bu bölümde yüksek skora ulaşmanın temel yolları şunlardır:

 Skor Kazanmanın Temel Yolları:

 1- Düşmanları ezmek veya ateş toplarıyla vurmak (özellikle koopa shell combo ile).

 2- Coin toplamak (her biri 200 puan).

 3- Power-up'lar almak (Mantar: 1000 puan, Fire Flower: 1000 puan).

 4- Gizli alanlara girmek (coin room vb.).

 5- Zıplarken düşmanlara seri şekilde çarpmak (combo).

 6- Level sonundaki bayrağa mümkün olan en yüksek noktadan atlamak (maksimum 5000 puan).

 🎮 Pseudocode – Super Mario Bros 1-1 En Yüksek Skor Stratejisi:

        START GAME
        LOAD Level 1-1

        INITIALIZE:
            score = 0
            combo_multiplier = 1
            position = start_point
            mario_state = small

        // ADIM 1: İlk Goomba'dan kaçın veya yok et
        IF Goomba detected THEN
            IF mario_state == small THEN
                jump over Goomba
            ELSE
                jump and stomp Goomba
                INCREASE score by 100

        // ADIM 2: Power-up bloğunu kır ve mantarı al
        MOVE to first visible question_block
        HIT block from below
        SPAWN mushroom
        CHASE mushroom until collected
        SET mario_state = big
        INCREASE score by 1000

        // ADIM 3: Coinleri topla
        FOR each coin_block in current segment:
            WHILE block still has coins:
                HIT block
                INCREASE score by 200

        // ADIM 4: İlk Koopa Troopa ile Shell Combo Yap
        FIND Koopa
        JUMP on Koopa
        KICK shell towards incoming enemies

        WHILE shell hits enemies:
            INCREASE combo_multiplier by 1
            INCREASE score by 100 * combo_multiplier

        // ADIM 5: Gizli coin odasına gir
        LOCATE hidden_pipe_near_first_gap
        IF pipe_enterable THEN
            ENTER pipe
            COLLECT all coins inside
            FOR each coin_collected:
                INCREASE score by 200
            EXIT pipe

        // ADIM 6: Fire Flower al (güvenli alanlarda)
        LOCATE second power-up block
        IF mario_state == big THEN
            HIT block to get Fire Flower
            COLLECT Fire Flower
            SET mario_state = fire
            INCREASE score by 1000

        // ADIM 7: Uzaktan düşmanları ateş ile vur
        FOR each visible enemy ahead:
            USE fireball
            IF hit THEN
                INCREASE score by 200

        // ADIM 8: Secret 1UP bloğunu al (ekstra can = skor potansiyeli)
        LOCATE hidden_1up_block_above_pipe_area
        JUMP to reveal and collect 1UP
        // not directly increasing score, but gives more lives for future

        // ADIM 9: Son bölümde zıplayarak bayrağa en yüksekten değ
        REACH end_flag
        JUMP to top of flagpole (timing and running)
        IF top_of_flagpole THEN
            INCREASE score by 5000
        ELSE
            INCREASE score by proportional height

        // OYUN BİTİŞİ
        DISPLAY final score
        END Level 1-1

 ## 🎮 Sorunun Cevabı:

 ### ❌ **Hayır, bu yöntem pratikte işe yaramaz** çünkü:

 ### 1. **Skor sıfırlanmaz ama hayat kaybı dezavantajlıdır**

 * Ölünce **skor sıfırlanmaz**, ancak her öldüğünde:

   * Checkpoint olmayan 1-1 gibi bir bölümde **baştan başlarsın**.
   * Bu düşmanlar zaten senin skorunu alabileceğin ilk seferde karşına çıkar.
   * Aynı düşmanları tekrar öldürerek daha fazla skor alırsın, **ama** bu skorlar önceki ile aynı kalır (katlanmaz), yani bir **çiftçilik (grind) stratejisi gibi işlemez**.

 ### 2. **1UP farming = sınırsız can, ama skor farming ≠ sınırsız skor**

 * Oyunda bazı noktalarda **1UP farming** (sonsuz can alma) yapılabilir (örn. shell stairs trick).
 * Ancak **skor farming**, bir noktadan sonra çok zorlaşır çünkü:

   * Aynı düşmanlardan tekrar tekrar puan almak **verimsiz** olur.
   * **Zaman kısıtlaması** (timer) nedeniyle aynı bölümü sürekli oynayarak yüksek skor yapmak **risklidir**.
   * Oyunun doğal akışında combo'lar gibi teknikler daha yüksek skor sağlar.

 ## 🧪 Uygulanabilir Bir Strateji (Pseudocode Kıvamında)

  ```pseudocode
  START Level 1-1

  WHILE lives > 0 AND player_wants_high_score:

      PLAY until checkpoint is reached

      // Checkpoint sonrası skor çiftçiliği başlasın
      LOOP:
          FIND Koopa
          DO shell combo if possible
          KILL Goombas and Koopas with fireball for 200 pts each
          COLLECT all reachable coins again
          DIE intentionally (fall in pit or hit enemy)
          DECREMENT lives by 1

          IF lives == 0:
              END game
          ELSE:
              RESPAWN at checkpoint

       // Eğer puan yeterliyse veya risk artarsa çık
       IF score > target OR lives low THEN
          EXIT LOOP

       FINISH level by jumping to top of flagpole
       INCREASE score by 5000 (if top)
   ```

 ---

 ### 💥 STRATEJİ: **"Checkpoint Farm Loop"**

 ```pseudocode
 INITIALIZE:
     total_score = 0
     lives = initial_lives (3 or more)
     time = 400

 PROGRESS to checkpoint in 1-1

 WHILE lives > 1:

     RESET time to 400

     // Skor toplama bölümü
     PERFORM shell combo with Koopa (e.g., in staircase)
     KILL Goombas and Koopas with fireball
     COLLECT coins and power-ups

     INCREASE total_score accordingly

     DIE intentionally (fall off cliff or hit enemy)
     DECREMENT lives by 1

     RESPAWN at checkpoint
     // Repeat

 // Son kalan can ile bölümü tamamla
 COMPLETE level
 JUMP to top of flagpole (5000 pts)
 INCREASE total_score by flag bonus
 ```
